Adding Iteration Metadata

Given

A Concordion specification includes an iteration meta tag in the head element, 

<head>
    <title>Adding Iteration  Metadata</title>
    <meta name="iteration" content="Grouping Tests"/>
</head>

When

The overview page is generated from AddingIterationMeta.html (this page),

Then

The iteration tag acts as an alias for the group tag (see Adding Group Meta) and so the generated overview page will list the test under a group heading. The generated overview page will include the Concordion run command along side the test, for example, 

<h2>Grouping Tests</h2>
<ul>
    <li>
        <a concordion:run="concordion" href="AddingIterationMeta.html">Adding Iteration Metadata</a>
    </li>
</ul>
