Markup a test as part of an iteration

An iteration can be used in place of a group in the HTML. This test belongs to iteration 1, it's configured in the HTML markup as shown when you 'view source'... 

    <head>
        <title>Markup a test as part of an iteration</title>
        <meta name="iteration" content="1"/>
        <link href="../style.css" rel="stylesheet" type="text/css"/>
    </head>
