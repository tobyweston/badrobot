Adding to Multiple Groups

Given

A Concordion specification includes a group meta tag in the head element with comma separated values, 

<head>
    <title>Adding Iteration  Metadata</title>
    <meta name="group" content="Grouping Tests, Multiple Groups"/>
</head>

When

The overview page is generated from AddingMultipleGroupMeta.html (this page),

Then

The generated overview page will be listed under multiple group headings. The generated overview page will include the Concordion run command along side the test only once to avoid duplicate runs.

The template can optionally indicate that the test is a duplicate, for example, using a class attribute. 

<h2>Multiple Groups</h2>
<ul>
    <li>
        <a class="ignored" href="AddingMultipleGroupMeta.html">Adding to Multiple Groups</a>
    </li>
</ul>
<h2>Grouping Tests</h2>
<ul>
    <li>
        <a concordion:run="concordion" href="AddingMultipleGroupMeta.html">Adding to Multiple Groups</a>
    </li>
</ul>
