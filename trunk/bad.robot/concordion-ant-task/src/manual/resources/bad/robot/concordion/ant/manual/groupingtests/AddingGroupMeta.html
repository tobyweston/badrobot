When you markup a Concordion specification as belonging to a group 

    <head>
        <title>Markup a test with a group</title>
        <meta name="group" content="1"/>
        <link href="../style.css" rel="stylesheet" type="text/css"/>
    </head>

The generated overview page will list the test under a group heading along with other tests belonging to the same group.

The generated overview page will include the Concordion 

run
command along side the test, for example, 

    <a concordion:run="concordion" href="src/manual/resources/bad/robot/concordion/ant/manual/groupingtests/AddingGroupMeta.html">Markup a test with a group</a>
